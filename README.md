# Particle Motion

An interactive particle animation built with **HTML5 Canvas** and **JavaScript** that demonstrates real-time rendering, dynamic motion, and browser-based graphics.

---

## Live Demo

🔗 https://evapatel123.github.io/particle-motion/


## Features

- Real-time particle animation
- Smooth motion using `requestAnimationFrame()`
- Dynamic rendering with the HTML5 Canvas API
- Lightweight and responsive
- Built with vanilla JavaScript

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- HTML5 Canvas API


##  About the Project

Particle Motion is an interactive web animation that simulates moving particles in real time using the HTML5 Canvas API. The project focuses on animation loops, rendering performance, and JavaScript object-oriented programming to create visually engaging motion effects.

##  Skills Demonstrated

- HTML5 Canvas
- JavaScript Animation
- Browser Rendering
- Object-Oriented Programming
- Event Handling
- Performance Optimization
